Now we know enough to use our favourite control statement - the if-else statement. This is used when we want a piece of code to run only if something happens. We can have as many else statements as we want, just put in another line saying elif (stands for else if). The syntax is
if condition:
code
elif condition:
code
else:
code

The situation - there are a bunch of people who are deciding whether to walk or ride bikes. Write a program that decides for them. If there are more people than bikes, they walk (otherwise it would just be unfair). If there are the same number of people as bikes, they ride.
